# Промпт: дизайн прототипа AR Farm

*Версия: 3.0 | Дата: 9 июня 2026*
*Изменения v3: убрана привязка к женской аудитории, нейтральный язык, «друзья» вместо «подруги»*

---

## Контекст проекта

Мобильная игра-ферма с реальными наградами. MVP — браузерный прототип (один HTML-файл). Продукт для российского рынка, запуск органикой. Целевая аудитория широкая. Ключевая дифференциация — возможность получить настоящий продукт (клубника, потом другие товары).

Core loop: **посадить → вырастить → собрать → обменять**.

---

## Аудитория и её ожидания

**Основной сегмент:** 25–45 лет, играющие в казуальные игры (Township, Hay Day, Family Farm). Сессии 5–15 минут несколько раз в день, ценят прогрессию и ощущение заботы о живых объектах.

**Дополнительный сегмент:** 18–35 лет, которым интересна геймификация и реальные награды (аудитория программ лояльности).

**Принципы для продукта:**
- Тёплая, уютная эстетика без агрессии — для всех
- Хочется расслабляться, а не тревожиться
- Управление одним пальцем
- Минимум текста, максимум визуального фидбека
- Прогрессия, забота о живых объектах

## Визуальный стиль

### Общий тон
Мягкий, тёплый, солнечный. Летний деревенский день. Ощущение: «я в безопасности и делаю что-то приятное».

**Не нужно:** пиксель-арт, тёмные тона, агрессивные акценты, неоновые цвета, киберпанк, минималистичный flat без текстур.

**Ориентиры:** Township, Stardew Valley (только тепло, без пикселей), Cozy Grove, акварельная иллюстрация.

## Экраны MVP

### 1. Сплеш-скрин
- Логотип «AR Farm» по центру — Nunito 900
- Анимация появления с лёгким «прыжком»
- Фон: тёплый градиент рассвета (жёлтый → голубой)
- Внизу: тонкий прогресс-бар загрузки
- Длительность: 2–3 секунды

### 2. Онбординг (4 слайда)

**Слайд 1** — «Вырасти свою ферму»
- Иллюстрация: огород на экране телефона
- Дружелюбное описание игры

**Слайд 2** — «Ухаживай и собирай»
- Клубника в фазах роста (семя → ягода)
- Механика полива и ухода

**Слайд 3** — «Получай настоящие продукты» ⭐
- Самый эмоциональный слайд — главный дифференциатор
- Подпись: «Игра в раннем доступе — доставка скоро»

**Слайд 4** — «Ферма лучше с друзьями»
- Делись урожаем — получай улыбки
- Кнопка «Пригласить позже» (мягкая, не блокирующая)

### 3. Главный экран — ферма

**Структура:**
```
[Хэдер: очки | прогресс | настройки | погода/время]
[Питомец у калитки]
[5 грядок с настроением растений]
[Дерево роста на заднем плане]
[Панель: Посадить | Полить | Статус | Подарить 🎁]
[Прогресс-бар: X / 500 очков]
[Уголок друзей]
```

**Грядки — 5 состояний:**
- Пустая — земля, иконка «+»
- Посеяна 🌱 — росток, таймер
- Росток 🌿 — побег
- Цветёт 🌸 — цветки
- Урожай 🍓 — ягоды, свечение

**Настроение растений:**
- Счастливое: светится, листья подрагивают, бабочка
- Нейтральное: обычное
- Ждёт внимания: листья поникли, цвет тусклее
- Предвкушение: пульсирует золотым за 30 мин до созревания

**Смена времени (реальное время устройства):**
- Утро (6–12): туман, роса
- День (12–18): яркое солнце, бабочки
- Вечер (18–22): закат, фонарики
- Ночь (22–6): тёмно-синий, светлячки

**Питомец-компаньон:**
- Маленький ёжик — постоянный NPC
- Idle: дышит, гуляет
- После сбора: радуется, сердечки
- После долгого отсутствия (24ч+): сидит у калитки, «Скучал… 🍓» — без штрафа

**Дерево роста:**
- 0–100: росток
- 100–300: деревце
- 300–500: цветущее
- 500+: большое с гнездом

**Ежедневный подарок:** коробка с лентой у калитки каждые 24ч

**Уголок друзей:** миниатюра фермы друга, «Друг собрал 20 ягод сегодня»

### 4. Туториал
- 3 шага: «Посади» → «Полей» → «Собери»
- Spotlight overlay, bubble-подсказка, палец 👆

### 5. Модалка сбора урожая
- Анимация «выпрыгивает»
- Вкладки: «Очки» / «Открытка»
- Кнопки: «Забрать» + «Отправить другу 🎁»
- Иногда — «золотая клубника» (confetti, переменное вознаграждение)

### 6. Статистика (bottom sheet)
- Урожаев собрано, очков всего, прогресс до награды
- Дерево роста — текущая фаза
- Лента событий
- Доска достижений

### 7. Настройки
- Звук, уведомления, «О проекте»

## Анимации

| Элемент | Анимация |
|---|---|
| Посадка | Земля взбугривается, росток появляется |
| Полив | Капли воды, синяя подсветка |
| Сбор | Ягоды разлетаются, очки летят вверх |
| Питомец idle | Дыхание, моргание |
| Питомец радость | Прыжки, сердечки |
| Питомец грусть | Поникшие ушки |
| Погода (дождь) | Капли по экрану, бонусный рост |
| Открытие подарка | Лента развязывается, сюрприз выпрыгивает |
| Золотая клубника | Confetti, питомец в восторге |
| Предвкушение | Мягкое золотое пульсирование |

## UX-принципы

1. **Без тревоги.** Нет штрафов — только замедление.
2. **Один палец.** Все действия — тап.
3. **Мгновенный фидбек.** Каждый тап → анимация.
4. **Прогрессия видна.** Прогресс-бар всегда на экране.
5. **Минимум текста.** Визуал первичен.
6. **Эмоциональная привязанность.** Питомец, настроение, дерево — живой уголок.
7. **Социальное тепло.** Без соревнования — обмен урожаем, совместные праздники.

---

## Тон голоса

- Разговорный, тёплый — как близкий друг, не навязчиво
- Без игрового жаргона
- Без давления («Успей!», «Не упусти!»)
- С заботой: «Твой урожай скоро созреет 🍓»
- Нейтрально по полу: «Отправить другу», «Друг заходил сегодня»

## Что не делать

- ❌ Тёмный фон
- ❌ Много текста на экране
- ❌ Tap-зоны менее 44px
- ❌ Анимации длиннее 400ms
- ❌ Рекламные баннеры
- ❌ Красные алерты и таймеры «успей»
- ❌ Лидерборды с позицией
- ❌ Штрафы за отсутствие
- ❌ Язык, ориентированный на один пол
