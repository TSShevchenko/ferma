# Промпт: дизайн прототипа AR Farm

## Контекст проекта

Мобильная игра-ферма с реальными наградами. MVP — браузерный прототип (один HTML-файл). Продукт для российского рынка, запуск органикой. Целевая аудитория — женщины 28–45 лет, играющие в Township, Hay Day, Family Farm. Ключевая дифференциация от конкурентов — возможность получить настоящий продукт (клубника, потом другие товары).

Core loop: **посадить → вырастить → собрать → обменять**.

---

## Аудитория и её ожидания

- Ценит тёплую, уютную эстетику без агрессии
- Хочет расслабляться, а не тревожиться
- Управляет одним пальцем
- Минимум текста, максимум визуального фидбека
- Любит прогрессию, декор, заботу о живых объектах

## Визуальный стиль

### Общий тон
Мягкий, тёплый, солнечный. Летний деревенский день. Ощущение: «я в безопасности и делаю что-то приятное».

**Не нужно:** пиксель-арт, тёмные тона, агрессивные акценты, неоновые цвета, киберпанк, минималистичный flat без текстур.

**Ориентиры:** Township, Stardew Valley (только тепло, без пикселей), Cozy Grove, акварельная иллюстрация.

### Палитра

| Роль | Цвет | Примерный HEX |
|---|---|---|
| Фон земли / грядки | Тёплый коричневый | `#8B5E3C` |
| Трава / газон | Насыщенный зелёный | `#5A9E3A` |
| Небо / задний план | Светло-голубой с жёлтым | `#E8F4C8` |
| Основной акцент (кнопки, прогресс) | Ягодно-красный | `#E8453C` |
| Дополнительный акцент (полив, вода) | Небесно-синий | `#4AADE8` |
| Текст основной | Тёмно-коричневый | `#3D2B1F` |
| Карточки / попапы | Кремово-белый | `#FFF8EE` |
| Очки / награда | Золотой | `#F5C542` |

### Типографика
- Заголовки: округлый, дружелюбный шрифт (Nunito, Comfortaa, Baloo 2)
- Тело: читаемый, не слишком тонкий (min 16px мобайл)
- Цифры / очки: чуть жирнее, чтобы выделялись
- Эмодзи-иконки как дополнение к тексту ✅

## Экраны MVP — требования к дизайну

### 1. Сплеш-скрин

- Логотип «AR Farm» по центру — рукописный или мягкий скругленный шрифт
- Анимация: логотип появляется с лёгким «прыжком» или нарастанием
- Фон: закат над полем или рассвет — тёплый градиент от жёлтого к голубому
- Внизу: тонкий прогресс-бар загрузки
- Длительность: 2–3 секунды, не блокирующая

### 2. Онбординг (3 слайда)

Слайд 1 — «Вырасти свою ферму»
- Иллюстрация: рука с телефоном, на экране телефона маленький огород
- Подзаголовок: короткое дружелюбное описание игры

Слайд 2 — «Ухаживай и собирай»
- Иллюстрация: анимированная клубника в фазах роста (семя → ягода)
- Подзаголовок: про механику полива и ухода

Слайд 3 — «Получай настоящие продукты» ⭐
- Иллюстрация: коробка с клубникой / купон / призовой попап
- Подпись мелким шрифтом: «Игра в раннем доступе — доставка скоро»
- Это слайд с главной дифференциацией, он должен быть самым эмоциональным

**Навигация:** буллеты/точки внизу + кнопки «Далее» / «Начать».

### 3. Главный экран — ферма

**Структура (сверху вниз):**

```
[Хэдер: монеты | очки | кнопка настроек]
[Поле с 5 грядками — основная зона]
[Панель действий: Посадить | Полить | Сеть]
[Прогресс-бар: X / 500 очков до разблокировки]
```

**Грядки:**
- 5 ячеек, 2–3 в ряд, расположены естественно, не в сетку
- Каждая грядка — прямоугольная перемычка тёмной земли
- Состояния:
  - **Пустая** — земля с лёгкой текстурой, иконка «+» или лейки
  - **Посеяна** 🌱 — маленький росток, таймер под грядкой
  - **Росток** 🌿 — заметный зелёный побег
  - **Цветёт** 🌸 — белые цветки на кустике
  - **Урожай** 🍓 — яркие красные ягоды, лёгкое свечение / анимация
- Таймер: маленький, под каждой грядкой, формат `0:32`
- При нажатии на спелую грядку — лёгкая вибрация + анимация сбора

**Задний план:**
- Небо, облачка, деревья / кусты по краям
- Деревянный забор или изгородь по периметру
- Ощущение пространства, не тесноты

### 4. Туториал (spotlight overlay)

- 3 шага: «Посади» → «Полей» → «Собери»
- Spotlight: полупрозрачная тёмная подложка, вырез вокруг активного элемента (мягкие скруглённые края)
- Bubble-подсказка: облачко рядом с вырезом, стрелка-указатель, иконка пальца 👆
- Текст подсказки: максимум 1–2 строки, разговорный тон
- Кнопка «Понятно» или автопрогресс по тапу

### 5. Модалка сбора урожая

- Попап по центру / bottom sheet снизу
- Анимация появления: «выпрыгивает» снизу или масштабируется
- Иллюстрация клубники крупно, с блеском
- Текст: «Урожай собран! +62 очка» (цифра анимирует, «подпрыгивает»)
- Кнопка «Забрать» — большая, заметная, цвет акцента
- После закрытия — анимация монеток / очков летит в хэдер

### 6. Экран прогресса / статистики (bottom sheet)

- Выезжает снизу по свайпу или кнопке
- Показывает: урожаев собрано, очков всего, прогресс до награды
- Полоска прогресса (500 очков = «разблокировать промокод» в будущем)
- Тон: мотивирующий, не статистический — «Отличная ферма! Ещё 213 очков до награды»

### 7. Настройки

- Простой экран: звук on/off, уведомления on/off, «О проекте»
- Без перегруза

## Анимации и микровзаимодействия

| Элемент | Анимация |
|---|---|
| Посадка | Земля «взбугривается», появляется росток |
| Рост фаз | Плавная трансформация иконки растения |
| Полив | Капли воды, синяя подсветка грядки |
| Сбор | Ягоды «разлетаются», очки летят вверх |
| Прогресс-бар | Заполнение с лёгким пружинным эффектом |
| Кнопки | Scale 0.95 при нажатии |
| Попапы | Spring easing, не линейный fade |

## Иконки и ассеты

**Растения клубники — 4 фазы:**
1. Семя в земле — маленький бугорок `🌱`
2. Росток — стебель с 2–3 листиками `🌿`
3. Цветок — куст с белыми цветками `🌸`
4. Урожай — куст с красными ягодами `🍓`

**Кнопки действий:**
- Посадить — иконка лейки / рассады
- Полить — иконка капли воды
- Статистика — иконка листа / дневника

**Хэдер:**
- Монеты — золотая монетка `🪙`
- Очки — звёздочка или ягода `⭐`

## Принципы UX

1. **Без тревоги.** Нет штрафов за пропущенный полив — только замедление. Нет "урожай сгнил полностью".
2. **Один палец.** Все действия — тап. Никаких свайпов для основных механик.
3. **Мгновенный фидбек.** Каждое нажатие дает видимый ответ — анимация, звук, вибрация.
4. **Прогрессия видна.** Прогресс-бар всегда на экране. Игрок понимает, куда идёт.
5. **Минимум текста.** Иконки и визуал первичны. Текст — только когда нельзя без него.

---

## Тон голоса (копирайт)

- Разговорный, тёплый, как подруга
- Без игрового жаргона ("XP", "Mana", "HP")
- Без давления ("Успей!", "Не упусти!")
- С заботой ("Твоя клубника скоро созреет 🍓")

---

## Что не делать

- ❌ Тёмный фон, мрачная атмосфера
- ❌ Много текста на одном экране
- ❌ Мелкие tap-зоны (менее 44px)
- ❌ Анимации длиннее 400ms для основных действий
- ❌ Рекламные баннеры, попапы с "купи сейчас"
- ❌ Нарушение ожиданий аудитории Township/Hay Day

---

## Приоритет экранов для дизайна

1. Главный экран (ферма с грядками) — самый важный
2. Модалка сбора урожая — ключевой эмоциональный момент
3. Онбординг слайд 3 (реальные продукты) — дифференциатор
4. Сплеш-скрин — первое впечатление
5. Туториал overlay — UX-критично
6. Прогресс / bottom sheet
7. Настройки
